An analogy to describe the differences between HTML and CSS

An analogy for the differences between HTML and CSS would be like an illustrated book. In the book we would see the contents i.e. the words and even the pictures in a book and CSS would be how you’d like the contents to to appear like the font, text-size and picture alignment.

The DOM is a model representation of for instance a webpage. This model allows us to interact with the page in a way that it can be accessed and manipulated e.g. JavaScript for events

Explain the difference between accessing data from arrays and objects.

Array are ordered lists of many objects. The Objects themselves may have many different properties by design. This means accessing an object in an array requires a specified index and furthermore the object you select may have different properties that need accessing itself.

Functions are great way to encapsulate repetitive code thus simplifying your code. It also helps by creating functionalities where you wouldn’t have to necessarily understand how the code works thus returning a solution to a complex problem.
